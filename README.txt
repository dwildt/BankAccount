To run this build file:
$ ant -f ccBuild.xml

Tools used: 
- Ant (ccBuild.xml)
- Emma/Jacoco (Code Coverage)
- PMD (Audit)
- CPD (Copy and Paste Detector)
- JUnit (Unit Testing)
- DDTests (Unit Testing)
- FIT (Acceptance Testing - Automated Documentation)
- Selenium Server (Functional Automated Testing)

After running the build, check the "out" folder, where you 
will find reports from junit, emma, pmd, cpd and fit.
